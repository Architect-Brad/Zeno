Features

🧠

Semantic NLU

Character n-gram intent classifier — 54 classes across 9 languages. Entity extraction, synonym expansion, multi-intent detection, dialog fragments. Pure Python, zero ML deps.

💻

Cross-Platform

Termux/Android, Windows, Linux. Voice I/O, notifications, volume, brightness, screen lock, app launcher. Each platform is a single provider file with mocked tests.

🔒

Zero Cloud

All processing on-device. No telemetry, no analytics, no API keys required. Weather via Open-Meteo (free, no key). Search via DuckDuckGo. RSS direct from sources.

🌐

9 Languages

English, Spanish, French, German, Hindi, Japanese, Korean, Portuguese, Arabic. Auto-detected from Unicode ranges. All 54 intents translated for each language.

🔄

LAN Sync

UDP broadcast discovery, REST push/pull, conflict resolution. No configuration, no mDNS, no cloud. Timers, profile, and conversation context sync across devices.

⏱️

Timers & Alarms

Threading-based scheduling, notification alerts, multi-turn slot-filling. Live countdown in web dashboard and TUI. Cancel or check status at any time.

🏠

Smart Home

Home Assistant REST API integration. Control lights, thermostat, door locks, security cameras. Configured via ~/.zeno/home_assistant.json.

🎤

Voice Modes

Push-to-talk, wake word, VAD-based native wake, continuous listening. termux-microphone-record or browser Web Speech API. Voice calibration per user profile.

🧩

Plugin System

Drop-in Python plugins in ~/.zeno/plugins/. Community registry with --discover. Extends BaseSkill — auto-discovered at import time. No registration needed.

🖥️

Web Dashboard

FastAPI + vanilla JS SPA. Chat interface, voice input, live timer panel, LAN peer discovery, profile editor, contacts API, settings panel. No build step required.

⌨️

Terminal UI

Textual-based TUI with chat log, input bar, live timer panel, status bar, and slash commands. Readline history, ANSI colors, tab completion.

⚡

Slash Commands

/w /t /a /v 75 /lights /s query — 20+ shortcuts. Shorthand syntax: 5m pizza → timer, 7am wake → alarm. All flow through the same NLU pipeline.

Quick Install

Prerequisites

Python 3.10+. Platform-specific tools optional.

# Termux/Android
pkg install termux-api
# Install Termux:API from F-Droid

# Linux
sudo apt install espeak-ng notify-send pulseaudio-utils brightnessctl

# Windows
# No additional tools needed (uses built-in PowerShell COM/WMI)

Install from source

git clone https://github.com/Architect-Brad/Zeno.git
cd Zeno/Code

# Core (no dependencies)
python -m zeno.core.runner

# Web dashboard
pip install "zeno[web]"
python -m zeno.web.server

# Terminal UI
pip install "zeno[tui]"
python -m zeno.core.runner --tui

Intent Classification

Character n-grams (size 2–4) with L2-normalized frequency vectors and cosine similarity:

  1. Text lowercased, stripped, padded with spaces
  2. Short texts (<8 chars) repeated 3× for sufficient features
  3. k-NN against ~5k vectors across 54 intents × 9 languages
  4. Word overlap bonus (+0.12) for non-stop-word matches
  5. Context boost (+0.03) during active slot-filling
  6. Ambiguity margin (0.06) triggers multi-intent mode
  7. Confidence threshold (0.30) rejects low-quality matches
  8. DDG fallback at marginal confidence (0.15–0.30)
