Zeno

An on-device voice assistant that runs entirely on your machine. Natural language understanding, cross-platform device control, LAN sync — with zero cloud dependencies.

🧠

Semantic NLU

Character n-gram intent classifier with synonym expansion and multi-intent detection. Understands 54 intent classes across 9 languages. Pure Python, zero ML dependencies.

💻

Cross-Platform

Runs on Termux/Android, Windows, Linux, and any terminal. Voice I/O, notifications, volume, brightness, screen lock — each platform provider is a single file.

🔒

No Cloud

Everything processes on-device. No telemetry, no analytics, no API keys required for core features. Weather uses Open-Meteo (free, no key). Your data never leaves your machine.

🌐

9 Languages

English, Spanish, French, German, Hindi, Japanese, Korean, Portuguese, Arabic. Auto-detected from Unicode ranges. All 54 intents translated for every language.

🔄

LAN Sync

Multiple Zeno instances discover each other via UDP broadcast. Context, profile, and timers sync via REST API. No configuration — works on any LAN.

⏱️

Timers & Alarms

Set timers, alarms, and reminders with natural language. Live countdown in the web dashboard and TUI. Cancel, check status, or set multiple simultaneously.

🏠

Smart Home

Control Home Assistant devices: lights, thermostat, door locks, security cameras. Speech-driven via REST API. Configured with a single JSON file.

🎤

Voice Modes

Push-to-talk, wake word, VAD-based native wake, and continuous listening modes. Works with termux-microphone-record on Android or browser Web Speech API.

🧩

Plugin System

Drop a Python file in ~/.zeno/plugins/ — it loads automatically. Community plugin registry with --discover. Each plugin is a BaseSkill subclass.

Quick Install

Core (no dependencies)
git clone https://github.com/Architect-Brad/Zeno.git
cd Zeno/Code
python -m zeno.core.runner
Web Dashboard
pip install "zeno[web]"
python -m zeno.web.server
Terminal UI
pip install "zeno[tui]"
python -m zeno.core.runner --tui

what time is it weather in London set a 5 minute timer called pizza turn on the lights 5m pizza /w London
